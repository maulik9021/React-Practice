09) Why we need keys in React? When do we need keys in React?
Ans: Key is a special string attribute you need to include when creating lists of elements in React. Keys are used in React to identify which items in the list are changed, updated, or deleted. Keys are used to give an identity to the elements in the lists.

10) Can we use index as keys in React?
Ans: The key is used by React to identify which elements have changed, which can help improve the performance of your application. However, a common mistake is to use the index of the element as the key . While this may seem like a reasonable solution, it can cause issues when the order of the list is changed.

12) What is a Config Driven UI ?
Ans: Config-driven UI is a technique that allows you to create user interfaces based on a configuration file, such as JSON, or a TypeScript file that defines the layout and content of the UI components. This can be useful for creating dynamic and customizable UIs without hard coding them.
