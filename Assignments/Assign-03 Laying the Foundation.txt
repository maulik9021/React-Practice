01) What is JSX?
Ans: JSX is an extended syntax to Javascript. It is used with React for designing UI. It is mostly HTML-like or XMl-like syntax. It is not HTML.


02) What are Superpowers of JSX?
Ans: Using JSX, you can write markup inside Javascript, providing you with a superpower to write logic and markup of a component inside a single.jsx file. It is easy to maintain and debug.


03) Role of type attribute in script tag? What options can i use there?
Ans: 'Type' attribute role in script tag is used to specifiy the type of the script. It is also used to specify the MIME type of script and identify the content of the Tag. It has default value which is "text/Javascript".


04) {TitleComponent} vs {<TitleComponent/>} vs {<TitleComponent></TitleComponent>} in JSX?
Ans: {TitleComponent()} vs {<TitleComponent/>} vs {<TitleComponent></TitleComponent>} all three of them are functional component.

It is rendered using all three ways above shown. It is at the end a normal javascript function.
